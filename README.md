## О приложении
Приложение предлагает вам информацию о случайно коктейле. Его фотографию, ингридиенты и их количество, а также рецепт.
Также сохраняются те коктейли, которые вы уже посмотрели и понравившиеся вам коктейли.

## Техническая часть
API взято с сайта https://www.thecocktaildb.com/api.php \
При запуске появляется экран загрузки, пока из интернета скачиваются данные о коктейле.\
UI сделан без storyboard с помощью стандартной библиотеки и SnapKit.\
Информация об ингридиентах и их количестве отображаеться в UITableView.\
Все просмотренные коктейли сохраняються в CoreData и отображаються в нижнем UICollectionView.\
При нажатии на кнопку обновить, появляеться loadingIndicator и загружается новый коктель.

При нажатии на кнопку "лайк", появляется Toast и в CoreData у данного Entity меняется флаг.\
При переходе на вкладку "любимые коктейли" отображеться UITableView, который заполняеться с NSFetchedResultController\
И при нажатии на коктейль осуществляеться переход на первый экран с отображением информации о коктейле.
